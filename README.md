# 🌸 Marin-chan

**Marin-chan** is the official Discord bot of **Manime Lounge**, designed to create a fun, welcoming, and safe community experience.

## ✨ Features

* 🛡️ Moderation tools
* 🌸 Welcome & goodbye messages
* ⭐ Leveling system
* 🎭 Reaction & auto roles
* 🎫 Ticket system
* 🎉 Giveaways & events
* 📊 Logging & server management
* 🎌 Anime-inspired features
* ⚙️ Utility commands
* 💬 Community engagement

## 🎯 Purpose

Marin-chan helps manage and enhance the Manime Lounge community by providing reliable moderation, useful utilities, and enjoyable features for all members.
